# NBTI 产品说明

NBTI 是一个移动端优先、可静态部署的魏晋南北朝人格趣味测试。它用 18 道情境题和 6 道五级量表校准题生成六维向量，再与 32 位历史人物做确定性距离匹配。

## 当前体验

- 首页：项目标题、开始入口和三幅静态人物视觉；不预先解释六维。
- 测试页：一次一道题、24 题进度、返回修改和清空重来。
- 结果页：主结果画像、用户人格文案、人物映照理由、可追溯史料、六维条、文字分享、海报、近邻和反向结果。
- 隐私：无后端、无统计、无 Cookie、无浏览器持久化；答案只存在于当前页面内存。

## 内容生产

人物文本采用 `docs/editorial/text-playbook.md` 的分批门禁。审美标定、三人试写和八人扩展未通过前，agent 草稿不得覆盖生产 JSON。史料采用结构化来源字段并在结果页提供原文链接。

## 静态部署

项目使用 Vite + React 和 hash 路由，可直接部署到 GitHub Pages。源 PNG 保留在 `assets-source/`，构建前生成轻量 WebP；完整部署产物设有 15 MB 预算。
